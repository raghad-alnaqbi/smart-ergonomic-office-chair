# Smart Ergonomic Office Chair System Project

## Project Overview

This academic project was completed as part of the Project Management course.

The project involved planning and managing the development of a smart ergonomic office chair using Agile methodology. Microsoft Project was used to create the project schedule, monitor progress, and manage project activities.

## My Role

- Participated in project planning.
- Used Microsoft Project for scheduling and monitoring.
- Collaborated on hardware integration, Bluetooth connectivity, mobile application development, and testing.
